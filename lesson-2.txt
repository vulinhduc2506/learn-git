1. git init: để cho project của mình vào work space
git add <file_name>: để cho một file có tên <file_name> di chuyển vào staging space sau khi có sự sửa đổi ở file đó
git add .: cho tất cả các file có sự chỉnh sửa trong project vào staging space
git commit -m 'message': di chuyển các sự thay đổi vào remote space
